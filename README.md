# 井字棋游戏

### 1.1 项目概述
基于虚幻引擎5.5的3D井字棋游戏，具有完整的UI、游戏流程和可扩展性。可通过配置系统调整棋盘大小（3×3到N×N）,ai 设置和胜利条件等。

### 1.2 功能列表

#### 1.2.1 核心游戏逻辑
- **棋盘系统**：支持可配置的N×N棋盘（默认3×3）
- **回合制机制**：玩家1和玩家2轮流落子
- **胜负判定**：检测横向、纵向、对角线的连珠胜利条件（可配置连珠数）
- **平局判定**：棋盘填满且无胜者时判定平局
- **计分系统**：记录双方胜场数
- **回合计时**：待完善
- **棋子撤销功能**：待完善
- **游戏存档系统**：待完善

#### 1.2.4 AI
- **难度选择**：
- Easy,Medium,Hard模式可以选
- **Easy**：
- 随机落子
- **Medium**：
- 判断威胁点择优落子+随机落子
- **Hard**：
- 判断威胁点择优落子+Minimax算法
- **算法实现**
- Minimax算法
- Alpha-Beta剪枝优化性能
- 棋盘hash积分缓存
- 启发式分数估算
